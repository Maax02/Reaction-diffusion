Reaction Diffusion Add-on (modularized)

Install this folder as a Blender add-on ZIP, or place the folder in Blender's addons directory.

Important:
- The compiled Cython module rd_core (.pyd on Windows) is already available in cython directory.
- NumPy and scikit-image can be installed from the add-on panel using the "Install RD Dependencies" button.
- Dependencies are installed into the local dependencies/ folder, not into Blender globally.
